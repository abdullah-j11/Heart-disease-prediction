04 / Project Overview

About This Project

Dataset

UCI Heart Disease dataset (Cleveland subset). 303 patients, 13 clinical features including demographic, physiological, and diagnostic measurements. Binary classification: presence or absence of heart disease (originally a 5-class problem, binarized to 0/1).

Pipeline

Data cleaning → EDA → Feature analysis → Train/test split (80/20, stratified) → StandardScaler for LR → Model training → Evaluation (accuracy, ROC-AUC, confusion matrix, sensitivity/specificity) → Feature importance.

Key Findings

  • Chest pain type was the strongest predictor (Gini importance: 0.31)
  • Asymptomatic chest pain — paradoxically — signals highest disease risk
  • Exercise-induced angina and ST depression are strong positive risk markers
  • Higher max heart rate is protective (younger, healthier cardiac response)
  • Decision Tree outperformed Logistic Regression on all metrics
